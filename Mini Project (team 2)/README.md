# 🌍 Climate Change Trend Analysis

A Data Science project that analyzes historical climate change data to identify trends in carbon dioxide (CO₂) emissions across different countries and years using Python.

BY:- Agastya 2420030208 and Jatin 2420030032

## 📌 Project Overview

Climate change is one of the most significant global challenges today. This project performs Exploratory Data Analysis (EDA) on historical climate data to uncover trends, patterns, and insights related to CO₂ emissions.

---

## 🎯 Objectives

- Analyze historical CO₂ emissions
- Perform data cleaning and preprocessing
- Visualize climate trends
- Compare emissions across countries
- Draw meaningful insights from the data

---

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## 📂 Dataset

Dataset Used:
**Our World in Data - CO₂ and Greenhouse Gas Emissions**

Dataset File:
`owid-co2-data.csv`

---

## 📊 Project Workflow

1. Import Libraries
2. Load Dataset
3. Data Cleaning
4. Exploratory Data Analysis (EDA)
5. Data Visualization
6. Trend Analysis
7. Key Findings
8. Conclusion

---

## 📈 Visualizations

- Global CO₂ Emissions Trend
- Top 10 CO₂ Emitting Countries
- Correlation Heatmap
- CO₂ Distribution
- CO₂ Per Capita Boxplot
- GDP vs CO₂ Scatter Plot

---

## 📌 Key Findings

- Global CO₂ emissions have increased significantly over time.
- Industrialized countries contribute the highest emissions.
- GDP and CO₂ emissions show a positive correlation.
- Population growth influences emission levels.
